A fully integrated blog module for PrestaShop e-commerce solution. It allows sellers to improve their stores by integrating a complete blog system.

Manage posts, create categories, let users add comments, and link your products with your posts in order to increase your catalog web traffic.

Give your visitors the opportunity to communicate with you, and provide an official information point on your store, with news, upcoming products, promotions, feedbacks, and so on ...

Many other possible uses :

  • Events
  • FAQ
  • News
  • Press Releases
  • Publications
  • ...

Features

Posts :

  • Title, summary, WYSIWYG content.
  • SEO (url rewriting, metas : title, description, keywords, sitemap xml).
  • Status (Published, Drafted, Suspended).
  • Publication date (delayed publication possible).
  • Images (unlimited) and main image selection, positions.
  • Link post to categories.
  • Link post to catalog products.
  • In product page, list related posts of product.
  • Enable / disable comments for post and moderation section.
  • Share with Addthis.
  • Search in posts feature.
  • Related posts.
  • RSS feed feature.
  • Multilingual & Multishop.
  • Pagination of the list of items.
  • Many configuration options.

Categories :

  • Add / edit / delete categories.
  • Name, description field.
  • SEO (url rewriting, metas : title, description, keywords).
  • Possible to see only category publications in front office.
  • Category position.
  • Enable / disable category.
  • Multilingual & Multishop.

Comments :

  • Moderate comments (status : "Pending", "Approved" or remove it).
  • Add / edit or remove comments.

Blocks :

  • Recent posts block (left or right column, or homepage content block).
  • Blog Categories block (left or right column).
  • Block search in articles.
  • Archives posts block.
  • Calendar articles search block.

Configuration :

  • Enable / disable categories, product relations, comments and url rewriting.
  • Image sizes.
  • Blocks Position.
  • Number of items per page and number of items in blocks.
  • Enable / disable popin for images.
  • Enable / disable comments moderation and allows or not anonymous comments.
  • Enable / disable share with Addthis
  • Enable / disable RSS
  • Etc...

Translations :

ES and EN translations included, but you can add your easily.
